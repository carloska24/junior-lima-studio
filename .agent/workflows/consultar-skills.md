# Workflow: Consultar Skills Antes de Implementar

// turbo-all

## 1. Identificar o tipo de trabalho

- Se for **frontend/design**: ler `.temp_ag_kit/.agent/skills/frontend-design/SKILL.md`
- Se for **mobile/responsividade**: ler `.temp_ag_kit/.agent/skills/mobile-design/SKILL.md`
- Se for **React**: ler `.temp_ag_kit/.agent/skills/react-patterns/SKILL.md`

## 2. Verificar agents relevantes

- Frontend: `.temp_ag_kit/.agent/agents/frontend-specialist.md`
- Mobile: `.temp_ag_kit/.agent/agents/mobile-developer.md`

## 3. Aplicar princípios da skill

- Preencher o CHECKPOINT da skill antes de codificar
- Evitar anti-patterns listados
- Seguir decision trees quando aplicável

## 4. Verificar qualidade

- Executar scripts de auditoria se disponíveis
- Testar no browser/dispositivo
